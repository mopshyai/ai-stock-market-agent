## RECORDING CHECKLIST

### Pre-Production
- [ ] Clean desktop (no icons, minimal distractions)
- [ ] Dashboard has real data (run fresh scan)
- [ ] Signal table shows variety of scores
- [ ] Telegram bot has test messages ready
- [ ] Charts folder has current images
- [ ] Music track selected (royalty-free)

### Production
- [ ] Record in 1080p or 4K
- [ ] Use clean cursor (hide if possible)
- [ ] Smooth mouse movements (no shaking)
- [ ] Consistent window sizing
- [ ] Good lighting (if showing face/phone)

### Post-Production
- [ ] Add text overlays (bold, readable font)
- [ ] Background music (30-40% volume, not overpowering)
- [ ] Voiceover recorded (clear audio, no background noise)
- [ ] Transitions smooth (avoid jarring cuts)
- [ ] Color grading consistent
- [ ] Export at 1080p, 60fps if possible
- [ ] File size under 100MB for easy sharing

## VOICEOVER TONE

**Style:** Confident but not aggressive
