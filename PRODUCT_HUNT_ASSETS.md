You're stuck manually checking charts, missing early signals, and making emotional decisions without a system.

- **Scans** 10-100+ stocks using advanced technical indicators (Bollinger Bands, RSI, ADX, EMA)

- **Scores** each signal from 1-8 (higher = stronger opportunity)

No manual work. No guessing. Just data-driven signals in your pocket.

## QUESTIONS YOU'LL GET ASKED (Prepare Answers)

### Q: How is this different from TradingView alerts?
**A:** TradingView requires manual setup for each stock/alert. We automatically scan your entire watchlist daily and rank opportunities by strength. You wake up to a curated list, not 100 alerts to sift through.

### Q: Does it actually work? What's the win rate?
**A:** Signals are based on proven technical indicators. We're building historical tracking so you can see performance over time. Not every signal wins (no system does), but it helps you focus on higher-probability setups instead of random picks.

### Q: Is it free?
**A:** Yes, it's open-source and self-hosted. You run it on your machine, so there's no subscription. Optional paid hosting + managed service coming soon for non-technical users.

### Q: Do I need to code?
**A:** For basic use, no. Just configure tickers in a YAML file and run it. The dashboard is point-and-click. Advanced customization requires Python knowledge.

### Q: What data does it use?
**A:** Yahoo Finance via the yfinance library. Free, reliable, covers US stocks, ETFs, crypto, and forex.

### Q: Can it trade automatically?
**A:** No. It only sends signals. You make all trading decisions. We're not building an auto-trader (legal/regulatory reasons).

### Q: Does it work for crypto?
**A:** Yes! BTC, ETH, and other major coins work if they're on Yahoo Finance.

### Q: How do I install it?
**A:** Clone the repo, install Python dependencies, configure your tickers, run the scan. Full guide in the README. Takes about 10 minutes.

### Q: Will you add [feature]?
**A:** Probably! What are you thinking? (Then engage with their idea)
